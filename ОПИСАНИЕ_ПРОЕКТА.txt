📊 ИНФОРМАЦИЯ О ПРОЕКТЕ
═══════════════════════════════════════════════════════════════════

Название:      Игра Мементо
Тип:           Декстопное приложение (Windows)
Язык:          Kotlin
Фреймворк:     Swing (Java GUI)
Сборка:        Gradle
Версия:        1.0


🎯 РЕАЛИЗОВАННЫЕ ФУНКЦИИ
═══════════════════════════════════════════════════════════════════

✅ Игровое поле 4×4 (16 карточек, 8 пар)
✅ Таймер - отсчет времени игры
✅ Счетчик попыток
✅ Красивая графика с вашими изображениями
✅ Анимация переворота карточек
✅ Проверка совпадений
✅ Автоматическое определение победы
✅ Кнопка "Новая игра"
✅ Статистика по завершению игры


🏗️ АРХИТЕКТУРА ПРОЕКТА
═══════════════════════════════════════════════════════════════════

1. Main.kt
   - Точка входа в приложение
   - Инициализация GUI

2. MemoryGame.kt
   - Основная логика игры
   - Управление состоянием
   - Таймер (использует Kotlin Coroutines)
   - Проверка совпадений
   - Подсчет попыток

3. MemoryCard.kt
   - Класс карточки
   - Визуализация (лицо/рубашка)
   - Состояния (открыта/закрыта/совпала)
   - Загрузка изображений

🎨 ОСОБЕННОСТИ ДИЗАЙНА
═══════════════════════════════════════════════════════════════════

• Темная цветовая схема (приятно для глаз)
• Анимация при переворачивании карт
• Цветовая индикация состояний:
  - Синий - закрытая карта
  - Зеленая рамка - найденная пара
  - Оранжевый статус - неверная попытка
  - Зеленый статус - правильная попытка

• Адаптивный размер карточек (120×120px)
• Отступы между элементами для лучшей читаемости


⚙️ ИГРОВАЯ МЕХАНИКА
═══════════════════════════════════════════════════════════════════

1. Старт игры:
   - Карточки перемешиваются случайным образом
   - Запускается таймер
   - Обнуляются попытки

2. Игровой процесс:
   - Игрок кликает на первую карточку → она переворачивается
   - Игрок кликает на вторую → переворачивается и проверяется
   - Если совпали → остаются открытыми (зеленая рамка)
   - Если нет → через 1 сек закрываются обратно
   - Попытка засчитывается после второго клика

3. Победа:
   - Все пары найдены
   - Таймер останавливается
   - Показывается статистика
   - Предложение начать новую игру


🔧 ВОЗМОЖНОСТИ НАСТРОЙКИ
═══════════════════════════════════════════════════════════════════

Вы можете легко изменить:

1. Размер поля (в MemoryGame.kt):
   gridSize = 4  →  4×4 = 16 карточек (по умолчанию)
   gridSize = 6  →  6×6 = 36 карточек (сложнее)

2. Добавить свои изображения:
   - Поместите JPG файлы в src/main/resources/images/
   - Назовите их: card1.jpg, card2.jpg, и т.д.

3. Изменить цвета:
   - Редактируйте Color() в MemoryGame.kt и MemoryCard.kt

4. Задержка при неверной попытке:
   - Timer(1000) в MemoryGame.kt → измените 1000 на другое значение (мс)
